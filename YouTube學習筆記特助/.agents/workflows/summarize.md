# 工作流：/summarize YouTube 影片一鍵摘要

當使用者輸入 `/summarize` 時，請執行以下步驟：

1. 檢查 `input/url.txt` 是否存在，並支援讀取多個 YouTube 網址（一行一個）。
2. 執行指令下載影片元數據與字幕：
   ```powershell
   python scripts/download_transcript.py
   ```
3. 讀取 `input/` 下產生的影片元數據與字幕檔。支援依據影片 ID 或播放清單資訊分別讀取。
4. 針對每個影片解析字幕內容，提煉核心重點並附帶 `[mm:ss]` 時間戳記。
5. **生圖判斷與訪談流程**：
   * **預設不生圖**：不要主動呼叫生圖工具。
   * **生圖訪談**：若使用者在此次對話中特別要求「需要生圖」，必須先暫停工作流，向使用者進行對話提問訪問（確認風格、色彩、比例），待使用者回答後再進行生圖，並將插圖儲存於 `output/images/`（或播放清單子資料夾的 `images/` 下）。
6. **分組與 Obsidian 同步**：
   * **播放清單分組**：若影片含有 `playlist_title`（播放清單標題），在 `output/` 與 Obsidian 的 `YouTube學習筆記/` 資料夾下，必須自動建立以該「播放清單標題」命名的子目錄，將筆記與圖片（如有）儲存於該子目錄中。
   * **儲存與同步**：生成最終筆記並儲存至本機的 `output/` 及 Obsidian 儲存庫的 **`YouTube學習筆記/`** 資料夾下。
7. 向使用者報告已生成，並附上專案 `output/` 與 Obsidian 的檔案連結與路徑。
