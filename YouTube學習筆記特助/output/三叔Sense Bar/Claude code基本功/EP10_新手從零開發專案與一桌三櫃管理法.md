## 🧠 核心概念與技術摘要
1. **「一桌三櫃」專案管理法**：
   * **一桌**：當前工作的專案主目錄（Workspace），保持乾淨，只放正在開發的核心代碼。
   * **三櫃**：
     - **設定櫃**：存放 API 密鑰、環境變數與設定檔（遠離 AI 隨手可得處）。
     - **備份/存檔櫃**：存放舊代碼備份與資源檔。
     - **文檔櫃**：存放需求書、使用說明與 API 文件（供 AI 隨時查閱）。
2. **從零開發教學應用程式**：
   * 示範如何遵循此管理法，使用 Claude Code 從空白目錄一步步建立出功能健全的備課工具。
