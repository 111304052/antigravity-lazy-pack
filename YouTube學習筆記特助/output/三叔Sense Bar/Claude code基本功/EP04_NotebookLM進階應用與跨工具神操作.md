## 🧠 核心概念與技術摘要
1. **五大進階應用情境**：
   * 教學簡報自動化、學生成績大數據分析、行政公文速讀與回信草擬、跨領域教材自動合成、以及知識庫自動同步。
2. **跨平台同步流 (Sync Pipeline)**：
   * 打造從 NotebookLM 自動同步重點到 Obsidian 筆記庫，並最終透過 Git 自動推送到 GitHub 的無感閉環工作流。
