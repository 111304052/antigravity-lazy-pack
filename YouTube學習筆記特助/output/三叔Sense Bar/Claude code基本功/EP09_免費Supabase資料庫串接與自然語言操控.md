## 🧠 核心概念與技術摘要
1. **Supabase 開源資料庫平台**：
   * Supabase 是極受歡迎的 Firebase 開源替代方案。它提供免費的 PostgreSQL 資料庫，並具備即時數據同步與認證功能。
2. **自然語言操控資料庫**：
   * 透過 Claude Code，使用者可以使用純中文/英文自然語言下達指令，AI 會自動在背景編譯 SQL 語句、建立資料表並存取資料，完全免除手寫資料庫語法的門檻。
