## 🧠 核心概念與技術摘要
1. **軍艦棋 (Battleship) 遊戲開發**：
   * 利用 Claude Code 在本地直接撰寫完整的 HTML5 軍艦棋雙人對戰遊戲，包括網頁結構、JavaScript 對戰邏輯。
2. **視覺優化威力展現**：
   * 使用 Claude Design 規範優化遊戲棋盤。利用毛玻璃卡片、雷達發光掃描微動畫、以及爆炸與擊中的漸層色回饋，將枯燥的格子圖改造為充滿科技感的對戰面板。
