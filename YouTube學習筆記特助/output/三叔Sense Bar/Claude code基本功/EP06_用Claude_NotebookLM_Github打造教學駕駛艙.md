## 🧠 核心概念與技術摘要
1. **教學駕駛艙 (Teaching Cockpit) 概念**：
   * 整合本機檔案（Obsidian）、雲端知識庫（NotebookLM）與雲端程式庫（GitHub）的總控面板，讓教師可以在單一界面完成備課、修改與上傳。
2. **工作流閉環實踐**：
   * 利用 Claude 作為大腦，自動讀取 NotebookLM 重點，生成教學網頁代碼，自動推送至 GitHub，學生即可即時存取最新教材。
