## 🧠 核心概念與技術摘要
1. **Agent 大軍 (Multi-Agent Swarm) 概念**：
   * 不再只用一個 AI 助理，而是免費組裝多個分工明確的 Agent 協作（如：1 個負責讀檔、1 個負責寫 CSS、1 個負責寫 JS、1 個負責測試）。
2. **多 Agent 協作管線**：
   * 透過 OpenCode 內部通訊協定，讓多個本機背景 Task 同步運行，自動完成複雜應用程式的編譯，並釋放單一對話的 Token 限制。
