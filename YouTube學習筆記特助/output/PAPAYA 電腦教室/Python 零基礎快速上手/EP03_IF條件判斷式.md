## 🧠 核心概念與技術摘要
1. **條件判斷結構 (Conditional Statements)**：
   * 使用 `if`、`elif`（else if 的縮寫）與 `else` 來根據不同條件執行對應的代碼塊。
2. **縮排與程式區塊 (Indentation)**：
   * Python 不使用花括號 `{}` 來區隔區塊，而是使用**「縮排（通常為 4 個空格或一個 Tab）」**。縮排錯誤會導致 `IndentationError`。
3. **比較運算子與邏輯運算子**：
   * 比較運算子：`==` (等於), `!=` (不等於), `>`, `<`, `>=`, `<=`。
   * 邏輯運算子：`and` (且), `or` (或), `not` (非)。
