## 🧠 核心概念與技術摘要
1. **For Loop (計數型迴圈)**：
   * 用於遍歷（Iterate）任何序列（如 List, String, range()）中的元素，逐個取出進行處理。
2. **範圍函式 `range()`**：
   * `range(stop)`：生成 0 到 stop-1 的數值序列。
   * `range(start, stop, step)`：生成從 start 開始，到 stop-1 結束，每次累加 step 的數值序列。
3. **迴圈控制陳述式**：
   * `break`：立刻強制終止並跳出整層迴圈。
   * `continue`：跳過本次剩餘代碼，直接進入下一次迭代。
