## 🧠 核心概念與技術摘要
1. **函式 (Function) 封裝**：
   * 將重複使用的代碼包裝成獨立區塊，提高程式的重複使用性（Reusability）與易讀性。使用關鍵字 `def` 宣告。
2. **參數與引數 (Parameters & Arguments)**：
   * 函式可接收外部傳入的參數，並在內部使用。
3. **回傳值 `return`**：
   * 使用 `return` 將運算結果傳回給呼叫端。`return` 會立刻終止函式的執行；若無寫 return 則預設回傳 `None`。
