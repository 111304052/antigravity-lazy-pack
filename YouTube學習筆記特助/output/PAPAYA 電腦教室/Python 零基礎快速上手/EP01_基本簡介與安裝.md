## 🧠 核心概念與技術摘要
1. **Python 的特色與應用**：
   * Python 是一種高階、直譯式、動態型態的程式語言。因其語法接近英文（簡潔易讀）、擁有豐富的第三方套件（如 NumPy, Pandas, TensorFlow 等），被廣泛應用於人工智慧、數據分析、網頁開發與自動化腳本。
2. **開發環境安裝 (Setup)**：
   * **Python 官網下載**：下載並安裝 Python，Windows 安裝時務必勾選「Add Python to PATH」以配置系統環境變數。
   * **IDE 選用**：推薦使用 VS Code（Visual Studio Code）作為程式碼編輯器，並安裝 Python 擴充功能以利偵錯。
3. **哈囉世界 (Hello World)**：
   * 學習第一行程式碼 `print("Hello World")`，理解直譯器如何逐行讀取並輸出結果。
