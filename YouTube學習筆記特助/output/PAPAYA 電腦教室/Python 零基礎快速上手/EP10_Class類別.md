## 🧠 核心概念與技術摘要
1. **物件導向程式設計 (OOP)**：
   * 是一種將資料與操作資料的方法封裝在「物件（Object）」中的編程範式。
2. **類別 (Class) 與物件 (Object)**：
   * **類別**：是製造物件的藍圖或模板（例如：「車子」類別）。
   * **物件**：是根據藍圖產出的實體（例如：「紅色賓士」物件）。
3. **建構子 `__init__` 與 `self`**：
   * `__init__` 是一個特殊的初始化函式，在建立物件時自動執行，用以定義物件的屬性（Attributes）。
   * `self` 代表物件實體本身，用於存取類別內的屬性與方法（Methods）。
