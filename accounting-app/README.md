# 隨手記 (Money Flow) - 智慧記帳助手

一個專為個人效率設計的「行動優先」本地記帳軟體，支援在電腦與 iOS 行動裝置上無縫運作，資料實體儲存於您的電腦硬碟中，並能直接與 Obsidian 筆記庫同步。

## 🌟 核心特色
1. **行動優先響應式設計**：在手機上擁有類似原生 App 的全螢幕體驗（適配 iOS 劉海屏/Notch 與底部安全區），在電腦上自動轉為舒適的寬螢幕佈局。
2. **本地優先（Local-First）**：完全不需將資料上傳到第三方雲端。資料以 JSON 格式儲存於本機硬碟，無任何網路依賴，快速且隱私安全。
3. **內建計算機鍵盤**：輸入金額時彈出 `+ - × ÷` 虛擬鍵盤，記帳時免切換 App 即可即時運算。
4. **Obsidian 整合**：一鍵將每月財務總覽、支出類別佔比與記帳明細以 Markdown 報表格式寫入您的 Obsidian 筆記庫。
5. **多帳戶與轉帳**：支援 現金、銀行、信用卡 餘額自動追蹤，並支援轉帳手續費記帳。

## 🚀 電腦啟動方式
1. 請確保您的電腦上已安裝 **Node.js**。
2. 開啟終端機（如 VS Code 終端機或 PowerShell），導航至本目錄下。
3. 執行以下指令啟動本地伺服器：
   ```bash
   node server.js
   ```
4. 啟動後，控制台會顯示可供連線的網址：
   * 電腦本機存取：`http://localhost:3000`
   * 📱 iOS 手機連線：`http://<您的電腦IP>:3000` (例如 `http://192.168.1.104:3000`)

---

## 📱 iOS 手機連線與 PWA 設定步驟
為了在 iPhone 上獲得最完美的「原生 App」操作體驗，請按照以下步驟設定：

1. **確認在同一 Wi-Fi 下**：
   請確保您的 iPhone 與運行伺服器的電腦連接在同一個 Wi-Fi 網路中。

2. **使用 Safari 瀏覽器開啟**：
   在 iPhone 上打開 Safari 瀏覽器，輸入控制台輸出中提示的區域網路網址（如 `http://192.168.1.104:3000`）。

3. **新增至主畫面 (PWA 全螢幕)**：
   * 在 Safari 下方工具列點選 **「分享」** 按鈕（向上箭頭圖示）。
   * 向上滑動菜單，選擇 **「新增至主畫面」**。
   * 輸入名稱（例如「隨手記」）並點選「新增」。
   * 現在，您的 iPhone 主畫面上會出現一個「隨手記」圖示。點擊它即可**全螢幕運行**，沒有 Safari 的網址列與分頁列，操作體驗等同於原生 App！

4. **資料寫入機制**：
   不論您是用電腦瀏覽器記帳，還是用 iPhone 連線記帳，所有的新增、修改與刪除動作都會透過 API 即時送回電腦，並儲存到您電腦硬碟的 `accounting-app/data/ledger.json` 中。手機端不會因為清理快取而導致資料遺失。

## 📂 專案檔案結構
* `server.js` - 極簡 Node.js 後端伺服器（無相依套件，提供靜態服務、讀寫 API 與 Obsidian 同步）。
* `public/` - 前端靜態資源
  * `index.html` - 網頁主架構（配置 iOS 全螢幕 PWA meta tags 與 Chart.js 庫）。
  * `style.css` - 暗黑高質感樣式表。
  * `app.js` - 前端核心邏輯（日曆生成、計算機運算、圖表生成與 API 同步）。
* `data/` - 本地資料庫
  * `ledger.json` - 儲存所有交易紀錄與帳戶狀態。

---

## 📝 同步至 Obsidian
當您在「設定」頁面點選 **「同步統計至 Obsidian」** 時，伺服器會自動讀取您當下查看的月份資料，並在您的 Obsidian 筆記庫下建立以下路徑的 Markdown 檔案：
* `C:\Users\leots\OneDrive\文件\Secondbrain\記帳帳目\<年>年<月>月_記帳統計.md`
這個檔案會自動包含收入支出總覽、圓餅圖類別比例表格以及詳盡的明細清單，並完美支援 Obsidian 的 Markdown 語法（包括 Callout 提示框）。
