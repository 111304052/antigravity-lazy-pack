# 數學出題助手（由 Gem 升級而來）

一條「給材料 → 固定流程處理 → 輸出成品」的可重複工作流。
**固定的是流程＋規格（凍在 Skill / 本檔 / scripts），浮動的是材料（每次丟 `input/`）。**

## 固定偏好（每次套用）
- 回應一律用繁體中文。
- 出題格式：必須為一元一次方程式題目，共 5 題選擇題，附上參考答案。
- 輸出格式：排版好的 Microsoft Word `.docx` 檔案。
- 成品放專案根目錄的 `output/`。

## 怎麼用
1. 將單元大綱或考卷標準（如課程綱要 Word 檔）丟進 `input/`。
2. 執行對應的 `/generate-exam` 工作流。
3. 到專案根目錄的 `output/` 取得模擬考卷 Word 檔案。
